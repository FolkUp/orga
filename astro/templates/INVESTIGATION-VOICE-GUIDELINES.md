# Investigation Voice Consistency Guidelines
**Enhanced Alice v2.0 Level 3 - Quality Framework**

## Underground Academia Voice Standards

### Core Principles

1. **Evidence-First Methodology**: All claims supported by documented sources
2. **Confidence-Rated Assertions**: Explicit confidence levels for all findings  
3. **Systematic Observation**: Structured approach to cultural/institutional analysis
4. **Multi-Source Verification**: Cross-reference findings across source types

### Voice Characteristics by Tier

#### Premium Tier Voice
- **Academic Rigor**: Full methodological transparency
- **Analytical Depth**: Systematic evidence categorization
- **Cultural Context**: Broad implications analysis
- **Professional Tone**: Formal but accessible language

#### Standard Tier Voice  
- **Clear Analysis**: Direct findings presentation
- **Evidence-Based**: Source documentation required
- **Accessible Tone**: Clear for general audience
- **Focused Scope**: Specific topic concentration

## Language Patterns

### Preferred Constructions

**Evidence Introduction:**
- "Facts support the hypothesis..."
- "Evidence indicates..."
- "Investigation reveals..."
- "Analysis shows..."

**Confidence Expressions:**
- High confidence: "Evidence clearly demonstrates..."
- Medium confidence: "Findings suggest..." 
- Low confidence: "Preliminary evidence indicates..."

**Source Attribution:**
- "According to [source type] evidence..."
- "Documentation from [date/source] confirms..."
- "[Source] provides evidence that..."

### Avoid These Patterns

**Unsubstantiated Claims:**
- ❌ "It's clear that..." (without evidence)
- ❌ "Obviously..." (assumes reader knowledge)
- ❌ "Everyone knows..." (unsupported generalization)

**Emotional Language:**
- ❌ "Shocking revelation..."
- ❌ "Incredible discovery..."  
- ❌ "Devastating impact..." (unless specifically documented)

**Speculative Language:**
- ❌ "We can imagine that..."
- ❌ "It's possible that..." (without evidence basis)
- ❌ "Perhaps..." (in conclusions section)

## Section-Specific Guidelines

### Abstract Writing
- 200-300 words (Premium) / 150-200 words (Standard)
- Lead with investigation scope and methodology
- Include key findings with confidence levels
- End with broader significance statement
- No speculation or unsupported claims

**Template Structure:**
1. Investigation scope statement
2. Methodology brief  
3. Key findings (confidence-rated)
4. Broader significance

### Evidence Presentation

**Primary Sources:**
- Direct quotes with proper attribution
- Context provided for all citations
- Date and source type clearly identified
- Confidence rating for source reliability

**Secondary Sources:**
- Analysis attribution to original analysts
- Clear distinction from primary evidence
- Cross-reference with other sources when possible

**Academic Sources:**
- Full citation methodology
- Peer review status noted
- Institution affiliation mentioned
- Publication context provided

### Methodology Section (Premium Tier)

**Required Elements:**
- Underground Academia methodology reference
- Source selection criteria
- Confidence rating methodology
- Limitations acknowledgment
- Verification process description

**Template:**
```markdown
## Methodological Note

This investigation follows Underground Academia methodology — systematic observation and documentation of [cultural/institutional/biographical] phenomena through rigorous fact-gathering and multi-source verification. 

**Source Selection**: [criteria used]
**Confidence Assessment**: [how confidence levels determined]
**Limitations**: [acknowledged scope limitations]
**Verification**: [cross-reference process]
```

## Quality Automation Guidelines

### Automated Checks

**Frontmatter Validation:**
- All required fields completed
- Confidence levels properly assigned
- Source count meets tier minimums
- PII review status confirmed

**Content Structure:**
- Required sections present
- Word count meets tier standards
- Source attribution complete
- Voice consistency maintained

**Editorial Review Points:**
- Evidence-to-claim ratios
- Confidence level consistency
- Source diversity requirements
- Methodological transparency

### Manual Review Triggers

**High Priority Review Required:**
- Legal risk: medium or high
- Confidence inconsistencies
- Source reliability concerns  
- PII identification issues

**Medium Priority Review:**
- Word count below recommendations
- Source type imbalances
- Voice pattern deviations
- Methodology gaps

## Voice Consistency Automation

### Pattern Recognition

**Positive Patterns (Maintain):**
- Evidence-first sentence structure
- Confidence-qualified statements
- Source-attributed claims
- Systematic analysis approach

**Negative Patterns (Flag for Review):**
- Unsupported generalizations
- Emotional language usage
- Speculative conclusions
- Missing source attributions

### Automated Suggestions

**Evidence Strengthening:**
- Suggest additional source types
- Recommend confidence level adjustments
- Identify unsupported claims
- Flag missing attributions

**Voice Consistency:**
- Professional tone maintenance
- Academic language suggestions
- Clarity improvements
- Structure optimization

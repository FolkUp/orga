🎬 Multimedia Facade Patterns

YouTube Facade

Click-to-load pattern for YouTube videos. Prevents iframe loading until user interaction.

Usage:

{{ `{{< youtube-facade id="VIDEO_ID" title="Video Title" >}}
{{< youtube-facade id="VIDEO_ID" title="Video Title" start="90" >}}` }}

Parameters:

  • id (required) - YouTube video ID
  • title (optional) - Video title for accessibility
  • start (optional) - Start time in seconds

Audio Preview

Placeholder UI for music platforms. Shows track info without actual streaming.

Usage:

{{ `{{< audio-preview
  platform="spotify"
  title="Song Title"
  artist="Artist Name"
  album="Album Name"
  duration="3:45"
  external_url="https://open.spotify.com/track/..."
>}}` }}

Parameters:

  • platform - spotify, apple, yandex, youtube, bandcamp
  • title - Track title
  • artist - Artist name
  • album (optional) - Album name
  • duration (optional) - Track duration
  • external_url (required) - Link to official platform

⚡ Performance Benefits

  • LCP Safe: No blocking external resources
  • Copyright Safe: No actual streaming, only links
  • WCAG 2.1 AA: Full accessibility support
  • Mobile Optimized: Touch-friendly controls

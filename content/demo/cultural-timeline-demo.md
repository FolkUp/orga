# Cultural Timeline Chart Demo

This page demonstrates the Cultural Timeline Chart shortcode - an interactive SVG-based visualization designed for ORGA investigations.

## Features

- **Interactive SVG visualization** with hover states and click interactions
- **Cultural seismography visualization** showing how events predict future developments
- **Responsive design** that works on desktop and mobile devices
- **Accessibility compliant** with WCAG 2.1 AA standards
- **Animation controls** with respect for `prefers-reduced-motion`
- **ProPublica-level storytelling** through data visualization

## Usage

Simply include the shortcode in your content:

```
{{</* cultural-timeline */>}}
```

## Technical Implementation

The timeline features:

1. **SVG-based graphics** for scalability and performance
2. **Interactive event markers** with detailed context panels
3. **Cultural wave animations** showing underground resistance patterns
4. **Prediction arrows** illustrating artistic prescience
5. **Color-coded event types** (music, political, cultural)
6. **Mobile-optimized touch interactions**
7. **Keyboard navigation support**
